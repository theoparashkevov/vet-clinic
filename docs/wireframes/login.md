# Login (Demo accounts v1)

Goals: fast access for demo doctor and owner; minimal friction; a11y-first.

Layout (mobile-first):
- Header: Vet Clinic logo
- Card: Email, Password, [Sign in] button
- Links: "Use demo doctor" / "Use demo owner" (prefill creds)
- Footnote: Language (EN), TZ shown as Europe/Sofia

States: loading, invalid creds, password visible toggle.

A11y: semantic form, labels/ids, input autocomplete, focus order, error summaries.
