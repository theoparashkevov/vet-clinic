# Owner Dashboard (supports multiple animals)

Top: Owner name, quick actions: [New appointment] [Add animal]

List: Animals (card per animal)
- Animal: name, species/breed, age, next appointment (if any)
- Actions: [View history] [Book appointment]

Upcoming appointments (owner): date/time, doctor, location, status; reschedule/cancel where allowed.

Empty state: guidance to add animal or book first appointment.

A11y: list semantics, button labels include animal name, high-contrast status chips.
